## ⚙️ הגדרות בסיסיות (חובה לשנות!)

פתח את `whatsapp-football-bot.js` ושנה:

```javascript
const CONFIG = {
    GROUP_NAME: 'כדורגל',  // 👈 שם הקבוצה שלך
    
    NAMES_TO_ADD: [
        'שמך',      // 👈 שנה לשמות האמיתיים!
        'חבר 1',
        'חבר 2'
    ],
    
    KEYWORDS: ['כדורגל', 'מגרש', 'יום'],
    REPLY_MODE: true,
    DELAY_MS: 2000
};
```
