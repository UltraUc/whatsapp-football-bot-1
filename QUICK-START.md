# ⚡ התחלה מהירה - 5 דקות להפעלה

## צעד 1: התקנת Node.js (אם עדיין לא מותקן)

### Windows:
1. לך ל-https://nodejs.org/
2. הורד את הגרסה LTS
3. הרץ את ההתקנה
4. בדוק שזה עובד: פתח CMD והקלד `node --version`

### Mac:
```bash
brew install node
```
או הורד מ-https://nodejs.org/

### Linux:
```bash
sudo apt update
sudo apt install nodejs npm
```

## צעד 3: התקן את החבילות

פתח טרמינל/CMD בתיקייה שבה שמרת את הקבצים והקלד:

```bash
npm install
```

זה ייקח 1-2 דקות. תראה הרבה טקסט - זה תקין!

---

## צעד 4: ערוך את ההגדרות

פתח את `whatsapp-football-bot.js` בעורך טקסט (Notepad, VS Code, וכו')

מצא את החלק הזה (שורות 6-28 בערך):

```javascript
const CONFIG = {
    GROUP_NAME: 'כדורגל',
    
    NAMES_TO_ADD: [
        'שמך הפרטי',  // 👈 שנה כאן!
        'חבר 2',
        'חבר 3'
    ],
    
    KEYWORDS: ['כדורגל', 'מגרש', 'יום'],
    
    REPLY_MODE: true,
    
    DELAY_MS: 2000
};
```

**שנה:**
1. `GROUP_NAME` - שם הקבוצה שלך (או חלק ממנו)
2. `NAMES_TO_ADD` - השמות שאתה רוצה שהבוט יוסיף
3. שמור את הקובץ!

---

## צעד 5: הרץ בדיקה (אופציונלי אבל מומלץ!)

```bash
node test-bot.js
```

אם הכל תקין, תראה:
```
✅ נמצאו 3 מקומות פנויים: 13, 14, 15
✅ נוספו 3 שמות לרשימה
```

---

## צעד 6: הפעל את הבוט!

```bash
npm start
```

### מה יקרה עכשיו?

1. **תראה QR code** - קוביות שחורות ולבנות בטרמינל
2. **פתח WhatsApp בטלפון:**
   - Android: שלוש נקודות ⋮ → Linked Devices → Link a Device
   - iPhone: הגדרות → Linked Devices → Link a Device  
3. **סרוק את הQR code**
4. **זהו!** הבוט עובד! 🎉

תראה הודעה:
```
✅ הבוט מוכן לפעולה!
```

## 🆘 משהו לא עובד?

### בעיה: "Cannot find module"
```bash
rm -rf node_modules
npm install
```

### בעיה: "QR code לא מופיע"
- בדוק חיבור לאינטרנט
- נסה לסגור ולפתוח את הטרמינל מחדש
- `npm start`

### בעיה: "הבוט לא מגיב לרשימות"
1. בדוק ש-`GROUP_NAME` נכון
2. בדוק ש-`KEYWORDS` מופיעות ברשימה
3. הרץ: `node test-bot.js` לבדיקה
